# Các Khái Niệm Cơ Bản Trong Machine Learning

## 1. Data (Dữ liệu)

Dữ liệu là thành phần cốt lõi của Machine Learning. Không có dữ liệu thì mô hình không thể học được gì cả.

### Các dạng dữ liệu:

#### 1. Định nghĩa (Structured Data)
- Dữ liệu có cấu trúc rõ ràng, được tổ chức theo dạng bảng
- **Ví dụ:**
  - Tập hợp thông tin khách hàng
  - Lưu dưới dạng bảng: số, chữ, hình ảnh

#### 2. Cấu trúc trong Excel
Dữ liệu thường được tổ chức theo 2 thành phần chính:

**1. Hàng (row):** Mẫu dữ liệu (học sinh, khách hàng...)
- Mỗi hàng đại diện cho một đối tượng/mẫu dữ liệu cụ thể
- Ví dụ: Một học sinh, một khách hàng, một giao dịch

**2. Cột (column):** Đặc điểm (Feature) hoặc Kết quả (Label)
- Mỗi cột chứa một thuộc tính/đặc điểm của dữ liệu
- Ví dụ: Tuổi, thu nhập, điểm số, giới tính

---

## 2. Chia Train/Test

Để đánh giá mô hình có học tốt hay không, ta cần chia dữ liệu thành 2 phần:

### 1. Training Set (Tập huấn luyện)
- Dùng để **huấn luyện** mô hình
- Mô hình sẽ học các pattern (mẫu) từ dữ liệu này
- Thường chiếm **70-80%** tổng dữ liệu

### 2. Test Set (Tập kiểm tra)
- Dùng để **đánh giá** hiệu suất mô hình
- Kiểm tra xem mô hình có dự đoán tốt trên dữ liệu mới không
- Thường chiếm **20-30%** tổng dữ liệu

**Lưu ý quan trọng:**
- Không được dùng Test Set để huấn luyện
- Test Set phải là dữ liệu "mới" mà mô hình chưa từng thấy

---

## 3. Thành Phần Học Có Giám Sát

Trong học có giám sát, dữ liệu được chia thành 2 phần chính:

### 1. Feature (Đặc trưng)
- Là các **thuộc tính đầu vào** của dữ liệu
- Dùng để mô tả đối tượng
- Ký hiệu: **X**

**Ví dụ:**
- Dự đoán giá nhà: diện tích, số phòng, vị trí
- Phân loại email spam: từ khóa, độ dài email, người gửi
- Nhận diện bệnh: tuổi, cân nặng, huyết áp

### 2. Label (Nhãn)
- Là **kết quả/đầu ra** mà ta muốn dự đoán
- Là câu trả lời đúng trong dữ liệu huấn luyện
- Ký hiệu: **y**

**Ví dụ:**
- Dự đoán giá nhà: giá (số tiền)
- Phân loại email: spam hay không spam
- Nhận diện bệnh: có bệnh hay không

### Tóm tắt: Data = Feature + Label

```
Dữ liệu hoàn chỉnh = X (Features) + y (Label)
```

**Trong quá trình:**
- **Training:** Mô hình học từ cả X và y
- **Prediction:** Mô hình chỉ nhận X, dự đoán ra y
